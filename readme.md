# Rust 学习进度记录

## 项目概述

这个仓库用于系统记录我学习《The Rust Programming Language》（Rust官方教程）的过程，包含各章节的练习代码和构思注释。

## URLs

- 官方书籍: [The Rust Programming Language](https://doc.rust-lang.org/book/)
- Rust 官方网站: [https://www.rust-lang.org/](https://www.rust-lang.org/)

## 目录结构

```
rust-practice/
├── rust_[order]_[title]/
│   ├── src/
│   ├── Cargo.toml
│   └── Cargo.lock
└── README.md
```
